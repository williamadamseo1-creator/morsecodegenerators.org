Frequently Asked Questions

How do I write a phone number in Morse code?

Convert each digit separately and keep spaces between each number symbol.

Is 0 the longest dash-only code?

Yes. Zero uses five dashes in international Morse.

Can I mix numbers and words in one message?

Yes. Use normal letter spacing and slash between words.
