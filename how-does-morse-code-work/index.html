How Morse Code Works: Timing, Spacing, and Structure

Morse code converts letters and numbers into timed dot and dash patterns. Timing rules control readability.

How to Use This Page

  1. Dot length = 1 timing unit, dash = 3 units.
  2. Gap inside one letter = 1 unit, between letters = 3 units.
  3. Word separation uses a larger gap, represented by slash in text mode.

Examples

  • A = .- (dot then dash)
  • S = ... (three dots)
  • O = --- (three dashes)
  • SOS = ... --- ...

Common Mistakes to Avoid

  • Using equal duration for dots and dashes.
  • Removing letter gaps and causing merge errors.
  • Treating slash as a letter instead of word separator.

Frequently Asked Questions

Why is Morse still readable with low bandwidth?

Because simple on-off timing patterns can pass through weak channels.

Is Morse code language-specific?

The code pattern system is standard, but mapped text depends on language set.

Do I need audio to understand timing?

No. You can start with text symbols and add audio later for speed.

Can I test timing patterns here?

Yes. Use converter output and compare to known examples like SOS and HELLO.
